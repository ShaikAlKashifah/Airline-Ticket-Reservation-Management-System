Airline Ticket Reservation System

Overview
This project is a full-stack Airline Ticket Reservation System developed using PHP and MySQL, designed to simulate the core functionalities of a real-world airline booking platform. The application allows users to search flights, book tickets, manage reservations, and enables administrators to manage flights and passengers efficiently.

This repository intentionally contains only documentation (README) and no source code, to demonstrate project understanding, system design, and implementation capability for academic and recruiter review purposes.

Key Features

User Features
- User registration and authentication
- Flight search based on source, destination, and date
- Airline ticket booking
- Seat reservation handling
- Booking confirmation and history

Admin Features
- Admin authentication
- Add, update, and delete flight details
- Manage schedules and seat availability
- View passenger booking details

System Features
- Secure form handling
- Database-driven architecture
- Dynamic data rendering using PHP
- Modular and scalable structure

Technology Stack
Frontend: HTML, CSS, Bootstrap
Backend: PHP
Database: MySQL
Server: Apache (XAMPP/WAMP)
OS: Windows / Linux

System Architecture
1. Client sends request via browser
2. PHP handles business logic
3. MySQL stores and retrieves data
4. Response rendered dynamically to UI

Database Design
Tables:
users – user account details
admin – admin credentials
flights – flight details
bookings – ticket booking records
passengers – passenger information
